Seawater Ultrafiltration System (S.U.S.)

Capstone Grade: A

This project is specified to treat seawater to get suitable water for Dyes Industry by using Ultrafiltration Membrane (UF), which is constructed from Cellulose Accetate and Nano-silicon as a filler. This project was successfully by maximizing the effect of Nano-silicon on the membrane pores. The membrane maximise the effeciency of water treatment with cost less than usual treatment

Tefnut's Pyramid

Capstone Grade: A

This project aims to solve the problem of Egyptian Shortage of water. This project addresses with this problem by collecting rainwater and use it in our demostic necessities. This project is an improvement of Permoda houses style to make tubes, which collects water. This project make its function with the least price and materials as it was constructed from wooden sticks and use on wooden sticks in our prototype water proof.
